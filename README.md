## 💡 安装教程

<details>
  <summary>展开/收起</summary>

#### ⚙️**获取令牌**

- [Github](https://github.com/settings/tokens)
- [Gitcode](https://gitcode.com/setting/token-classic)

#### 🔧Yunzai 根目录执行命令安装

- Github

```bash
git clone --depth=1 https://用户名:令牌@github.com/Lovely-02/Y-Plugin.git ./plugins/Y-Plugin/
```

> [!NOTE]
> 如果你的网络环境较差，无法连接到 Github，可以使用 [02](https://github.com/Lovely-02) 提供的文件代理加速下载服务
>
> ```
> git clone --depth=1 https://用户名:令牌@ghproxy.521002.xyz/https://github.com/Lovely02Y/Y-Plugin.git ./plugins/Y-Plugin/
> ```

- Gitcode

```bash
git clone --depth=1 https://用户名:令牌@gitcode.com/Lovely-02/Y-Plugin.git ./plugins/Y-Plugin/
```

#### 🔧 安装依赖

```bash
pnpm install
```

#### 🔧 安装禁漫

- 禁漫依赖

```bash
pip3 install img2pdf jmcomic
```

- 加密依赖

```bash
sudo apt-get install qpdf
```

- 如需添加 ck 请在 `./resources/jmpdf/option.yml` 中添加如下内容
- AVS 从 [18comic.vip](https://18comic.vip/) 中获取
- 更多配置前往 [JMComic](https://github.com/hect0x7/JMComic-Crawler-Python/blob/master/assets/docs/sources/option_file_syntax.md)

```bash
client:
  postman:
    meta_data:
      proxies: system
      cookies:
        AVS: 84227871
```
